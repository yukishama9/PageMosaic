## What is PageMosaic?

PageMosaic is a visual static-site editor that lets you manage reusable components, multi-language content, and themes — all without a build framework. Edit a navbar once, and every page that uses it updates automatically.

Whether you're running it as a desktop Electron app or in your browser with a simple Python server, PageMosaic keeps your HTML projects organized and maintainable.
